# Beastsite 2.0

*Adapted from https://github.com/nordicgiant2/vue-nice-homepage*

## Overview

Each page on the website is created via a `README.md`. For instance, `docs/README.md` generates the homepage.

## Viewing the site

### Local

0. Navigate to the root of this repository (`beastsite/`).

1. Run `npm install`.

2. Run `npm run dev`.

3. Visit [http://localhost:8080](http://localhost:8080) and watch magic happen :)

### On website

First, complete steps 0-3.

4. Run `npm run build` to generate necessary HTML/JavaScript files under `dist/`.
